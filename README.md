# KasirKita 🛒
Sistem kasir sederhana berbasis web — by Hunar

## Fitur
- ✅ Kasir dengan keranjang belanja & struk digital
- ✅ Manajemen produk (tambah, edit, hapus)
- ✅ Laporan penjualan harian & produk terlaris
- ✅ Alert stok hampir habis
- ✅ Tampilan gelap modern

## Tech Stack
- React + Vite
- Deploy: Vercel / GitHub Pages

## Cara Jalankan Lokal
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
```
